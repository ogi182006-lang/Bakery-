# Images Directory

This directory is for local bakery images.

The website currently uses Unsplash images via CDN URLs.
To use your own images:

1. Add your image files here (e.g., chocolate-cake.jpg)
2. Update the `src` attributes in `index.html` to reference:
   `images/your-image-name.jpg`

Recommended image sizes:
- Hero image: 600×600px
- Product cards: 400×300px
- Gallery items: 400×400px (large: 800×450px)
- About section: 500×700px
